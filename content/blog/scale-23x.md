## What is SCALE?
SCALE is North America's largest community-run open-source conference, running since 2002. It's a great
way for those interested in Linux and FOSS to gather and talk about the latest news. Topics range from
open-source infrastructure and kernel developments to desktop software demos like GIMP or Kdenlive.

## Who should go?
Anyone can join! It's a wonderful environment, especially for new Linux users. You'll learn a ton and
discover new things about the ever-changing scene.

## What if I'm a BSD user?
Come anyway! We'll talk about the kernel together :^)
